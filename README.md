# On The Land Lawn Service Website

Premium lawn care website for On The Land Lawn Service, serving Lehigh Acres, Fort Myers, and Cape Coral, Florida.

## Deploy to Vercel

### Option 1: One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_GITHUB_USERNAME/landscaping-website)

### Option 2: Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Navigate to project
cd landscaping-website

# Deploy
vercel

# Deploy to production
vercel --prod
```

### Option 3: Git Integration

1. Push to GitHub/GitLab
2. Import project at [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects Next.js
